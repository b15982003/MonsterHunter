# Features 功能特色!

### 首頁
* 可以看到所有使用者發的文章，點進去可以看到詳細的文章內容，對於文章的一切可以留言發表意見。


<img src="https://i.imgur.com/1a3DmgL.jpg" width="220" height="390"></a>
<img src="https://i.imgur.com/QpAVQ9b.png" width="220" height="390">

### 個人頁面
* 個人頁面可以看到你自己登入的照片以及名稱，參戰的次數以及對所有武器的熟悉度，熟悉度越高獲得的星星數量也就越多，當你集滿的時候武器的 icon 就會亮起。

<a>
<img src="https://i.imgur.com/bVV7YzK.png" width="220" height="390">
</a>



### 歷史訊息
* 在裡面可以看到你自己所有對戰紀錄的成功或是失敗，你隊友拿的武器以及你們所花費的時間。

<a>
<img src="https://i.imgur.com/tRrvtLH.png" width="220" height="390">
</a>



### 聊天室
* 在這裡可以看到所有約戰的房間，看到有自己會玩的武器也可以毫不猶豫加入，在裡面可以聊天以及輔助魔物獵人這款遊戲，計算遊戲時間以及任務成果，最後可以評分所有隊友以達到客觀的評分。
*  擴音系統 擴音功能讓 google 小姐幫你大聲喊出來操控隊友

### 好友頁面
* 可以看到輸入好友 email 搜尋好友，點選你有興趣的使用者可以看到他對所有武器的熟練度
，覺得有很厲害的人或是朋友還可以按追隨進入你的朋友名單，點到追隨者就可以看到所有你追隨的人，隨時都能觀看他的進度。
